BAKE LOG

Welcome to the (work-in-progress state of) Baker's Logbook! Over the course of the past 6 months, I have been consistently documenting my baking journey in a small notebook. It contains recipes that I've repeatedly practiced, as well as notes on what worked and what didn't. My goal is to digitize this notebook and create a more refined "logbook" that I can easily reference and share with others. It also serves the purpose of learning frameworks I haven't worked with before. I've been using React professionally for the past couple of years, so I chose to build this project using Angular. The styling is achieved using Tailwind CSS, and the backend is powered by Firebase.

This site is currently work-in-progress. Features may be incomplete or subject to change. Your feedback is very much appreciated -- please feel free to report any bugs or leave suggestions here.

Welcome to the Baker's Logbook! Over the past seven months, I have been consistently documenting my baking journey in a small notebook. It contains recipes that I've repeatedly practiced, as well as notes on what worked and what didn't. My goal is to digitize this notebook and create a more refined "logbook" that I can easily reference and share with others. This project is being built using Angular and styled with Tailwind CSS, while recipe data and user auth are managed using Firebase Authentication and Cloud Firestore.
